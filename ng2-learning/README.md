Angular 2.0 learning project
=============================

1.  Install
	*  chmod 777 ./install.sh (unix/mac)
	*  ./install.sh (unix/mac)
	*  ./install.bat (windows)

2.  Start Type script compiler
	*  npm run tsc

3.  Start server
	*  npm start

4.	Open http://localhost:8080 in browser to view the application.
