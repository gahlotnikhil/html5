Angular 2.0 learning project
=============================

1.  Install
	*  chmod 777 ./install.sh
	*  ./install.sh

2.  Start Type script compiler
	*  chmod 777 ./startTSC.sh
	*  ./startTSC.sh

3.  Start server
	*  chmod 777 ./startServer.sh
	*  ./startServer.sh

4.	Open http://localhost:8080/client/html/ in browser to view the application.
